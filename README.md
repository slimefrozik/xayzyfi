# xayzify! site

## Что внутри
- index.html — основной лендинг
- styles.css — стили и визуальная система
- script.js — анимации при скролле
- press-kit.md — текстовый пресс‑кит
- media-kit.json — структурированные данные для медиа
- humans.txt — команда и контакты
- robots.txt — правила индексации
- sitemap.xml — карта сайта
- manifest.webmanifest — PWA-метаданные

## Быстрый старт
Откройте index.html в браузере.
